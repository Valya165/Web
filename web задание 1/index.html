История создания Python

Python был создан в 1991 году Гвидо ван Россумом в центре математики и информатики в Нидерландах. Язык был задуман как преемник языка ABC, способного к обработке исключений и взаимодействию с операционной системой Amoeba.

Название языка произошло вовсе не от вида пресмыкающихся. Автор назвал язык в честь популярного британского комедийного телешоу 1970-х «Летающий цирк Монти Пайтона».

В 1994 году вышел Python 1.0. В состав версии 1.0 вошли функциональные программные инструменты lambda, map, filter и reduce. Ван Россум описал философию ядра языка в документе «Руководство по стилю кода Python» (Python Style Guide), позже названном PEP 8.

Философия Python

Тим Петерс сформулировал 19 принципов философии Python в документе «Дзен Python» (PEP 20). Вот некоторые из них:

  • Красивое лучше, чем уродливое
  • Явное лучше, чем неявное
  • Простое лучше, чем сложное
  • Сложное лучше, чем запутанное
  • Читаемость имеет значение

Версии и развитие

Ключевые моменты в развитии языка:

  1. Python 2.0 (2000)
    • Введение сборщика мусора
    • Поддержка Юникода
    • Понимание списков (list comprehension)
  2. Python 3.0 (2008)
    • Несовместимые изменения с Python 2
    • Улучшенная поддержка Юникода
    • Изменение синтаксиса print

Основные характеристики

Python — высокоуровневый язык программирования общего назначения с следующими ключевыми особенностями:

  • Динамическая типизация
  • Автоматическое управление памятью
  • Полная интроспекция

Базовые конструкции

            
# Пример функции
def greeting(name):
    """Простая функция приветствия"""
    return f"Привет, {name}!"

# Работа со списками
numbers = [1, 2, 3, 4, 5]
squares = [x**2 for x in numbers]  # Списковое включение

# Классы в Python
class Animal:
    def __init__(self, name):
        self.name = name
    
    def speak(self):
        pass

class Dog(Animal):
    def speak(self):
        return f"{self.name} говорит Гав!"

# Использование
dog = Dog("Бобик")
print(dog.speak())  # Выведет: Бобик говорит Гав!
